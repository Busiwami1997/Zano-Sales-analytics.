# ZANO E-Commerce Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive **Power BI Sales Dashboard** developed to analyze e-commerce performance, customer behavior, and regional sales trends.  
The dashboard enables quick executive-level insights into revenue performance, top customers, and geographic opportunities.

## Dashboard Preview
![Dashboard Screenshot](zano-ecommerce-dashboard.png)

## Key Insights Delivered
- Total Revenue, Total Orders, and Average Order Value KPIs
- Sales trend analysis over time
  - Top 10 cities generating the highest sales
- Category and performance comparisons

## Tools & Technologies
- Power BI Desktop
- SQL (Data extraction and transformation)
- Data Modeling (Fact and Dimension tables)
- DAX Measures for KPIs and aggregations

## Business Value
This dashboard helps decision-makers:
- Detect revenue trends quickly
- Understand geographic sales performance
- Support data-driven strategic planning
